Update 3.3
 Meet new connection option! FlexiHub keeps getting better! Today, we are happy to announce the latest update to FlexiHub for Windows.

The new version adds lots of minor improvements as well as the new outstanding feature called “Keep connections active”. The new option will let you stay connected to your remote devices even if your computer temporarily lost the Internet connection, or the device was unplugged from the server machine for a few minutes. For more details, see FlexiHub user guide.

The new connection option is now available in the Options menu.
